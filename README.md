# app-react-cmontecinoa

Aplicacion desarrollada con React como parte de mi proceso formativo en desarrollo frontend.

## Demo

URL publica esperada en GitHub Pages:

`https://cmontecinoa.github.io/app-react-cmontecinoa/`

## Descripcion

Este proyecto presenta una interfaz simple de ecommerce construida con React. Incluye una barra de navegacion, tarjetas de productos y una estructura base de componentes reutilizables.

## Tecnologias utilizadas

- React
- JavaScript
- HTML
- CSS
- Bootstrap

## Funcionalidades

- Barra de navegacion con carrito
- Vista de productos con tarjetas
- Estructura de componentes separada
- Build listo para despliegue en GitHub Pages

## Scripts disponibles

En el directorio del proyecto puedes ejecutar:

### `npm start`

Inicia la app en modo desarrollo.

### `npm run build`

Genera la version de produccion en la carpeta `build`.

### `npm test`

Ejecuta las pruebas configuradas por Create React App.

## Despliegue

El repositorio incluye un workflow de GitHub Actions para publicar automaticamente en GitHub Pages al hacer push a la rama `master`.
